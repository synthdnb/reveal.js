The Philosophy of Ruby

Ruby is designed to make programmers happy.

가끔 사람들은 코드를 만들기 전에 생각을 정리하기 종이에 수도코드를 작성하죠.

만약에 그 수도코드가 그대로 컴퓨터에서 작동한다면 어떨까요?

저는 제가 하고 싶은 일에 집중하기를 원하지,
언어의 public static void main...같은 걸 신경쓰고싶지 않아요.

그렇기 때문에 루비를 간결한 언어로 만들었습니다.

Block Syntax 


def func
	return yield(10)
end

func do |x|
	return x+3
end

return 안써도 마지막 Evaluation 결과를 알아서 리턴해요


def func
	yield(10)
end

func do |x|
	x+3
end

do end로 묶지 않고 이렇게 쓸수도 있어요


def func
	yield(10)
end

func{ |x| x+3 }
